# medjs

Official client-side JavaScript library for the [medibloc](https://github.com/medibloc/go-medibloc) blockchain.

## Install

### Node.js

```bash
npm install medjs
```

## Usage

```javascript
var Medjs = requrie('medjs');
var medjs = Medjs(['http://localhost:9921']);
```
